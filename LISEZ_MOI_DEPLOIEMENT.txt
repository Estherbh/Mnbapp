================================================================================
                        COMMENT DEPLOYER ?
================================================================================

METHODE 1 : AUTOMATIQUE (RECOMMANDE)
------------------------------------

1. Allez sur : https://console.cloud.google.com/
2. Creez un projet : "virunga-dashboard"
3. Cliquez sur l'icone Cloud Shell (>_) en haut a droite
4. Dans le terminal qui s'ouvre, tapez :

   git clone https://github.com/Estherbh/Mnbapp.git
   cd Mnbapp
   bash deploy.sh

5. Suivez les instructions a l'ecran
6. C'est tout ! Votre app sera en ligne en 5-10 minutes


METHODE 2 : MANUELLE (ETAPE PAR ETAPE)
---------------------------------------

Suivez le guide : GUIDE_DEPLOIEMENT_GCP.md

Ce guide contient toutes les etapes detaillees avec des explications.


METHODE 3 : VISUELLE (AVEC ILLUSTRATIONS)
------------------------------------------

Suivez le guide : GUIDE_VISUEL_GCP.md

Ce guide contient des diagrammes pour chaque etape.

================================================================================
                        FICHIERS A CREER SUR GCP
================================================================================

Certains fichiers ne sont pas sur GitHub pour des raisons de securite.
Vous devez les creer manuellement sur GCP :

1. users.json
   - Contient les utilisateurs et leurs mots de passe
   - Utilisez generate_hash.py pour creer les hash
   - Exemple disponible dans : users.json.example

2. Fichiers Excel (a uploader)
   - COLLECTE DES DONNEES TERRAIN_RELATIONS EXTERIEURES (2).xlsx
   - Revue de la presse2.xlsx
   - VNP LOGO FRENCH.jpg
